# Fluo Colors

Fluo Color Palette should typically be reserved for data visualizations and artworks. These colors are less strict on [WCAG 2.0](https://www.w3.org/TR/UNDERSTANDING-WCAG20/visual-audio-contrast-contrast.html) accessibility standards and should therefore not be used for typical user interface design.
